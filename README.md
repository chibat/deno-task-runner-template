# Deno Task Runner Template

This is Task Runner Template Repository for Deno.

This repository does not use Third Party Tools.

## Usage

### display runnable tasks

```
$ ./tasks.ts
build
test
dev
start
deploy
```

or

```
$ deno run tasks.ts
build
test
dev
start
deploy
```

### run task

```
$ ./tasks.ts start
```

or

```
$ deno run -A tasks.ts start
```
